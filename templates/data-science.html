Data In, Insight Out

Wrangling raw CSVs into clean datasets, visualizing trends across multiple chart types, processing images with NumPy, and predicting prices through linear regression.

First, Clean the Data

Before any chart or model means anything, the data has to be trustworthy. Using a century of Nobel Prize records, this project explores, audits, and cleans a raw dataset — checking what's missing, what's duplicated, and what needs fixing before any real analysis can begin.

  • Loaded a real-world dataset of 962 Nobel Prize records using pandas, then explored its shape, columns, and structure.
  • Checked for duplicate entries and audited every column for missing (NaN) values to understand the data's overall quality.
  • Identified and substituted missing birth dates for laureates with incomplete records, documenting each assumption made.
  • Used value_counts() to uncover patterns, like finding every repeat Nobel Prize winner in the dataset's history.
  • Engineered a new prize_share percentage column by parsing and converting string-based fraction data (e.g. "1/2") into usable numeric values.

Then, Let the Data Speak

A clean dataset is only half the story — the other half is knowing how to look at it. Across these projects, I've leaned on visualization to ask better questions before reaching for any model: what's correlated, what's skewed, where the outliers hide, and what the data actually looks like once it's no longer just rows and columns. Each chart is chosen for what it can reveal, not just how it looks.

  • Built regression plots (regplot) to visualize the relationship between movie production budgets and worldwide gross, complete with a fitted trend line and confidence interval.
  • Used displot with kde to explore the distribution of housing features like number of rooms, spotting skew and outliers in the data.
  • Generated a pairplot across multiple variables (NOX, RM, LSTAT, DIS, PRICE) to compare every relationship at once and form quick hypotheses before modeling.
  • Combined scatter and distribution views with jointplot to study how pollution levels (NOX) relate to distance from employment centers (DIS).
  • Mapped Nobel Prize counts by country using a Plotly choropleth map, and built a horizontal bar chart to rank the top 20 countries by category.
  • Plotted a line chart with a rolling average over scatter data to smooth out yearly noise and reveal the long-term trend in prizes awarded.

Then, Compute Like an Array

Underneath every chart and every image is just a grid of numbers — and once you can manipulate that grid, you can manipulate almost anything. This project explores NumPy's ndarray from the ground up: building vectors and matrices, understanding their shape and dimensions, and using them to perform real computation, including treating images themselves as arrays to transform and recolor.

  • Explored 1D, 2D, and N-dimensional arrays, learning how shape, dimensions, and indexing change as data scales from vectors to tensors.
  • Practiced array slicing and generation with tools like arange() and linspace() to create and extract subsets of data precisely.
  • Performed vector arithmetic — addition, multiplication, and broadcasting scalars across arrays — to understand how NumPy handles element-wise operations.
  • Implemented matrix multiplication using both .matmul() and the @ operator, verifying results by hand against the standard linear algebra definition.
  • Treated a real image as a multi-dimensional array, applying transformations like rotation and color inversion directly through array operations.
  • Converted a color image to grayscale by normalizing pixel values and applying a weighted RGB-to-luminance formula, then rendered the result with Matplotlib.

Then, Predict the Price

With the data understood and visualized, the next step is turning it into a working model. This project builds a multivariable linear regression to predict home prices from thirteen different features, then goes further than just fitting a model — testing whether transforming the target variable improves accuracy, sense-checking the coefficients against real-world intuition, and validating performance on data the model has never seen.

  • Split the dataset into training and test sets to evaluate model performance on unseen, out-of-sample data rather than the data it was trained on.
  • Trained a multivariable linear regression using scikit-learn, combining 13 features into a single pricing equation and scoring it with R-squared.
  • Extracted and interpreted the model's coefficients, sense-checking their signs and magnitudes against real-world expectations — like whether more rooms genuinely raises price.
  • Analyzed residuals by plotting predicted values against actual prices to visually assess how well the model's predictions track reality.
  • Applied a log transformation to the target variable to correct skew, then retrained the model and compared the new residual distribution and R-squared against the original.
  • Used the trained model to estimate the value of custom properties by feeding in specific feature values and reversing the log transformation to get a dollar estimate.
