Professional background

I'm a professional accountant and financial analyst, with experience in financial oversight, invoice scrutiny, and cost estimation within the real estate development sector. My work involves close attention to numerical accuracy, contract compliance, and identifying discrepancies before they become costly problems.

Technical journey so far

Over the past several months, I've been building toward a second skill set in software development and data analysis. Through Python, I've built a full-stack web application with Flask and SQLAlchemy, complete with user authentication and a database-driven CRUD system. I've used web scraping with BeautifulSoup and Selenium to automate real-world tasks — from compiling a personal movie ranking database using a live API, to auto-generating Spotify playlists from historical Billboard charts, to building bots that handle gym class bookings and apartment listing submissions without manual input. On the data side, I've worked extensively with pandas, NumPy, and visualization libraries like Seaborn and Plotly to clean, explore, and analyze real datasets — uncovering patterns, handling missing data, and building basic linear regression models to predict outcomes like pricing trends.

Future direction

Looking ahead, I want to bring these two worlds together — using software development and data analysis to solve real problems in finance and accounting. One long-term goal is building a Financial Management Information System (FMIS) that brings more transparency, automation, and real-time oversight to processes I currently handle manually, like invoice verification and budget tracking. I'm also increasingly drawn to artificial intelligence — not just using AI tools, but eventually contributing to how they're built, whether through automation systems, intelligent data pipelines, or applied machine learning. This portfolio is a running record of that transition: from analyzing numbers by hand to building the systems that can analyze them for me. .
