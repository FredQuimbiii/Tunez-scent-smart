Automate It, Then Build the Interface

From AI bots that book gym slots and hunt for apartments, to desktop apps built with Tkinter — a quiz game, a password manager, and a U.S. states guessing game — these projects turn repetitive tasks into automation and ideas into interactive tools.

Scrape, Fill, Submit, Repeat

Apartment hunting usually means hours of copying listing details into spreadsheets by hand. This project automates the entire pipeline — scraping live listings, then driving a real browser to fill out and submit a form for every single result, no manual data entry required.

  • Scraped a live property listing site using BeautifulSoup to extract prices, listing URLs, and addresses for every available apartment.
  • Used Selenium WebDriver to control a real Chrome browser, simulating natural user interaction rather than just reading raw HTML.
  • Configured anti-detection browser settings — disabling automation flags and using a persistent Chrome profile — to keep the session stable and undetected.
  • Automatically filled out and submitted a Google Form for each listing, looping through every scraped apartment without manual input.
  • Built a reusable Form class with dynamic field-mapping logic, so each form submission correctly matched location, cost, and link data to the right input field.

Never Miss a Spot

Popular gym classes fill up in seconds — by the time you remember to book, the spot's gone. This bot logs in, finds the exact class by date and time, and books it instantly, automatically joining the waitlist if it's already full. No more racing to refresh the page.

  • Automated full login authentication using Selenium, handling email and password fields without any manual interaction.
  • Used explicit waits (WebDriverWait) to reliably locate dynamically loaded class cards by their date-specific element IDs.
  • Built smart conditional booking logic — if a class is full, the bot automatically joins the waitlist instead of failing outright.
  • Tracked real-time stats across runs, including classes booked, waitlists joined, and total classes processed.
  • Generated a clean booking summary report printed to console after each run, so I always know exactly what happened.

Know Your States

Memorizing all 50 U.S. states is easier when it's a game. This project uses Python's Turtle graphics library to turn a blank map into an interactive quiz — type a state's name, and it writes itself directly onto the correct spot, live score and all.

  • Used Turtle graphics to render a blank U.S. map and write state names directly onto their correct geographic coordinates.
  • Loaded state names and their x/y coordinates from a CSV file using pandas, matching each guess against real map data.
  • Built a live scoring system that updates in real time, showing progress out of all 50 states as you play.
  • Tracked guessed states in a list to prevent duplicate entries and accurately measure progress.
  • Added an "Exit" command that automatically exports every missed state to a new CSV file — turning the game into a built-in study list.

Two Apps, One Toolkit

Two desktop apps, same toolkit. A password manager that generates and securely stores credentials, and a trivia quiz that pulls live questions from the web — both built with Python's Tkinter to turn simple scripts into real, interactive programs.

  • Built a Password Manager with a Tkinter GUI, using Canvas and PhotoImage to add a custom logo and a clean, structured grid layout.
  • Generated strong, randomized passwords by mixing letters, numbers, and symbols, then shuffled the result for true randomness.
  • Automatically copied each generated password to the clipboard using pyperclip, so it's instantly ready to paste.
  • Added input validation and confirmation dialogs with messagebox, preventing empty submissions and confirming details before saving to a local file.
  • Built a Trivia Quiz app using clean object-oriented design — separate classes for the question model, quiz logic, and user interface, each handling a single responsibility.
  • Pulled live trivia questions in real time from the Open Trivia Database API, decoding HTML entities with Python's built-in html module for clean display.
  • Tracked score and question progress through a dedicated QuizBrain class, checking user answers against the API's correct response on each round.
  • Gave instant visual feedback — the canvas flashes green or red on each answer — before automatically advancing to the next question using Tkinter's after() method.
